# Key Remapper

Press one key to automatically press another key.

Created by **BANANCHIKIREAL**.

[View the source code on GitHub](https://github.com/BANANCHIKIREAL/KeyRemapper).

## Example

The default remap is:

**Up -> Tab**

When you press Up, the mod also presses Tab. When you release Up, the mod
releases Tab.

Your original Up key still works normally.

## How to use

1. Open the settings for Key Remapper.
2. Choose the **Trigger Key** you want to press.
3. Choose the **Also Presses** key the mod should send.
4. Enable the places where the remap should work.

All changes apply immediately.

## Where it works

You can enable or disable the remap separately in:

* Levels and Practice Mode
* The Level Editor
* Menus

## Indicator

While playing a level, the indicator shows your current remap.

You can:

* Show or hide it
* Move it with **Indicator X** and **Indicator Y**

The indicator flashes when the remap activates.

## Important

* **Enabled** turns remapping on or off without deleting your settings.
* The trigger key is not blocked; the output key is added to it.
* Do not set both keys to the same key.
* This mod is made for Windows.
